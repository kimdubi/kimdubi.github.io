PostgreSQL background process мӮҙнҺҙліҙкё°

мӢңмһ‘н•ҳл©°

PostgreSQLмқҳ background processлҠ” кё°ліё н”„лЎңм„ёмҠӨмқё Postmasterм—җм„ң fork() лҗң н”„лЎңм„ёмҠӨл“Өмһ…лӢҲлӢӨ.
к°Ғ fork() лҗң background processл“ӨмқҖ DBMS мҡҙмҳҒмқ„ мң„н•ҙ к°Ғмһҗ л§ЎмқҖ м—ӯн• мқ„ мҲҳн–үн•ҳкІҢ лҗҳлҠ”лҚ°мҡ”.
м§ҖлӮң кёҖм—җм„ң PostgreSQLмқҳ architectureм—җ лҢҖн•ҙ мӮҙнҺҙліҙл©° к°„лһөн•ҳкІҢ л§җм”Җл“ңл ём§Җл§Ң
мқҙлІҲ кёҖм—җм„ңлҠ” PostgreSQLмқҳ background processл“Өм—җ лҢҖн•ҙ мўҖ лҚ” мһҗм„ёнһҲ мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ.

PostgreSQL background process

Postmaster

Postmaster лҠ” лӘЁл“  background processл“Өмқ„ м ңм–ҙн•ҳкі  client => DBMSлЎңмқҳ connectionмқ„ мғқм„ұн•ҳкі  лӢ«м•„мЈјлҠ” мөңмҙҲмқҳ н”„лЎңм„ёмҠӨмһ…лӢҲлӢӨ.

  • DB кё°лҸҷ мӢң PostmasterлҠ” лӢӨлҘё background processлҘј fork() н•ҳлҠ” м—ӯн• кіј
  • clientк°Җ PostgreSQL DBMSм—җ connectionмқ„ мҡ”мІӯн•ҳл©ҙ PostmasterлҠ” м„Өм •лҗң мқёмҰқл°©лІ•(pg_hba.conf)м—җ л”°лқј мӮ¬мҡ©мһҗлҘј мқёмҰқн•ҳкі 
    мқёмҰқмқ„ нҶөкіјн•ҳл©ҙ мӮ¬мҡ©мһҗм—җкІҢ м„ңл№„мҠӨлҘј м ңкіөн•ҳкё° мң„н•ң мғҲлЎңмҡҙ sessionмқ„ fork() н•ҳлҠ” м—ӯн• мқ„ н•©лӢҲлӢӨ.

BgWriter

BgWriter н”„лЎңм„ёмҠӨлҠ” Shared Memory лӮҙ dirty pageлҘј л””мҠӨнҒ¬м—җ лӮҙл Өм“°лҠ” н”„лЎңм„ёмҠӨмһ…лӢҲлӢӨ.

  • shared bufferмқҳ dirty pageлҘј л””мҠӨнҒ¬лЎң мЈјкё°м ҒмңјлЎң flush мҲҳн–ү
  • мЈјкё°м Ғмқё checkpoint л°ңмғқ мӢң лӘЁл“  dirty pageлҘј л””мҠӨнҒ¬м—җ кё°лЎқ

мқҙ мӨ‘ мІ«лІҲм§ё мһ‘м—…мқҖ dirty pageлҘј лҜёлҰ¬ мЎ°кёҲм”© flush н•ҙлҶ”м„ң checkpoint лҸҷмһ‘ л•Ң I/Oк°Җ лӘ°лҰ¬м§Җ м•ҠлҸ„лЎқ н•ҳмһҗ
мҰү, мӢңмҠӨн…ң I/O load мқҳ л¶ҖлӢҙмқ„ мӨ„мқҙкё° мң„н•ң лҸҷмһ‘мқҙлқјкі  ліј мҲҳ мһҲмҠөлӢҲлӢӨ.

кҙҖл Ё parameter

#bgwriter_delay = 200ms                 # 10-10000ms between rounds
#bgwriter_lru_maxpages = 100            # max buffers written/round, 0 disables
#bgwriter_lru_multiplier = 2.0          # 0-10.0 multiplier on buffers scanned/round
#bgwriter_flush_after = 512kB           # measured in pages, 0 disables
  • bgwriter_delay : л‘җ лІҲмқҳ м—°мҶҚм Ғмқё flush лҸҷмһ‘ к°„ delay м„Өм •
  • bgwriter_lru_maxpages : bgwriterк°Җ н•ңлІҲм—җ лӮҙл Өм“°лҠ” лҚ°мқҙн„° лӢЁмң„лЎң dirty pageк°Җ 100 ліҙлӢӨ лҚ” л§Һмңјл©ҙ м„ңлІ„ н”„лЎңм„ёмҠӨм—җм„ң мІҳлҰ¬лҗЁ

WalWriter

Write Ahead Loggingмқҳ к°ңл…җмқҖ лҚ°мқҙн„° нҢҢмқјмқҳ мҲҳм •мқҖ ліҖкІҪ мӮ¬н•ӯмқҙ лЎңк·ём—җ лЁјм Җ кё°лЎқ лҗң нӣ„м—җл§Ң мқҙлЈЁм–ҙм ём•јн•ңлӢӨлҠ” кІғмһ…лӢҲлӢӨ.
мқҙ л©”м»ӨлӢҲмҰҳмқ„ мӮ¬мҡ©н•ҳл©ҙ л””мҠӨнҒ¬м—җ лҚ°мқҙн„°лҘј мһҗмЈј лӮҙл Ө м“°м§Җ м•Ҡкі  disk I/OлҘј мӨ„мқј мҲҳ мһҲмҠөлӢҲлӢӨ.
лҳҗн•ң WAL лЎңк·ёлҘј мӮ¬мҡ©н•ҳм—¬ dbms лҘј мөңк·ј мӢңм җк№Ңм§ҖлҸ„ ліөкө¬н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

кҙҖл Ё нҢҢлқјлҜён„°

#------------------------------------------------------------------------------
# WRITE-AHEAD LOG
#------------------------------------------------------------------------------

# - Settings -

wal_level = logical                     # minimal, replica, or logical
                                        # (change requires restart)
#fsync = on                             # flush data to disk for crash safety
                                        # (turning this off can cause
                                        # unrecoverable data corruption)
#synchronous_commit = on                # synchronization level;
                                        # off, local, remote_write, remote_apply, or on
#wal_sync_method = fsync                # the default is the first option
                                        # supported by the operating system:
                                        #   open_datasync
                                        #   fdatasync (default on Linux)
                                        #   fsync
                                        #   fsync_writethrough
                                        #   open_sync
#full_page_writes = on                  # recover from partial page writes
#wal_compression = off                  # enable compression of full-page writes
#wal_log_hints = off                    # also do full page writes of non-critical updates
                                        # (change requires restart)
#wal_init_zero = on                     # zero-fill new WAL files
#wal_recycle = on                       # recycle WAL files
#wal_buffers = -1                       # min 32kB, -1 sets based on shared_buffers
                                        # (change requires restart)
#wal_writer_delay = 200ms               # 1-10000 milliseconds
#wal_writer_flush_after = 1MB           # measured in pages, 0 disables

#commit_delay = 0                       # range 0-100000, in microseconds
#commit_siblings = 5                    # range 1-1000
  • wal_level : WAL logfileм—җ кё°лЎқлҗҳлҠ” м •ліҙмқҳ мҲҳмӨҖ, кё°ліёк°’мқё replica л¶Җн„° streaming replication кіј hot standbyлҘј мӮ¬мҡ©н•  мҲҳ мһҲмқҢ
  • fsync : ліҖкІҪмӮ¬н•ӯмқ„ WAL logfile лЎң лӮҙл Өм“°кІ лӢӨлҠ” м„Өм •, off м„Өм • мӢң м„ұлҠҘ мғҒ мқҙм җмқҙ мһҲмңјлӮҳ crash л°ңмғқн•ҳл©ҙ лҚ°мқҙн„° мҶҗмӢӨлӮЁ
  • synchronous_commit : ON м„Өм • мӢң ліҖкІҪмӮ¬н•ӯмқҙ WAL logfile м—җ кё°лЎқлҗҳм–ҙм•ј нҠёлһңмһӯм…ҳ мҷ„лЈҢ
  • wal_sync_method : ліҖкІҪмӮ¬н•ӯмқ„ WAL logfileлЎң лӮҙл Өм“°лҠ” л°©лІ•, fsync, fdatasync л“ұмқҙ мһҲмқҢ
  • full_page_writes : м „мІҙ data pageлҘј WAL logм—җ м“°лҸ„лЎқ м„Өм •
  • wal_buffers : WAL logлҘј м ҖмһҘн•ҳлҠ” лҚ° мӮ¬мҡ©лҗҳлҠ” л©”лӘЁлҰ¬ кіөк°„, wal_writer_delay, commit_delay м„Өм •м—җ мҳҒн–Ҙл°ӣмқ„ мҲҳ мһҲмқҢ
  • wal_writer_delay : walwriter н”„лЎңм„ёмҠӨмқҳ лҸҷмһ‘ к°„кІ© м„Өм •мңјлЎң, мӢңк°„мқҙ л„Ҳл¬ҙ кёёл©ҙ WAL bufferк°Җ л¶ҖмЎұн•  мҲҳ мһҲмңјл©° мӢңк°„мқҙ л„Ҳл¬ҙ м§§мңјл©ҙ WALмқ„ кі„мҶҚ м“°кІҢлҗҳм–ҙ л””мҠӨнҒ¬ I/O л¶ҖлӢҙмқҙ мҰқк°Җн•  мҲҳ мһҲмқҢ
  • commit_delay : commit мқҙнӣ„ WAL flush лҗҳкё° м „ WAL bufferм—җм„ң лҢҖкё°н•ҳлҠ” мӢңк°„, group commit кіј к°ҷмқҖ нҡЁкіј
  • commit_siblings : commit delay м„Өм •мқ„ мҲҳн–үн•ҳкё° м „, н•„мҡ”н•ң мөңмҶҢ лҸҷмӢң нҠёлһңмһӯм…ҳ мҲҳ, нҠёлһңмһӯм…ҳ мҲҳк°Җ commit_siblings ліҙлӢӨ м ҒмқҖ кІҪмҡ° л°”лЎң WAL flush лҗЁ.

PgArch н”„лЎңм„ёмҠӨ

WAL logfileмқҖ мһ¬нҷңмҡ© лҗҳкё° л•Ңл¬ём—җ кіјкұ°мқҳ WAL logifleмқҖ мғҲлЎңмҡҙ logлЎң лҚ®м–ҙ м”Ңм—¬м§‘лӢҲлӢӨ.
мӨ‘к°„м—җ WAL logfileмқҙ мң мӢӨлҗҳлҠ” кІҪмҡ° мӣҗн•ҳлҠ” мӢңм җмңјлЎң мӢңм җліөкө¬к°Җ м–ҙл Өмҡё мҲҳ мһҲкё° л•Ңл¬ём—җ PostgreSQLм—җм„ңлҠ” WAL logfileмқ„ л°ұм—…н•ҙлҶ“мқ„ мҲҳ мһҲлҠ”лҚ° мқҙл•Ң м—ӯн• мқ„ мҲҳн–үн•ҳлҠ” н”„лЎңм„ёмҠӨк°Җ PgArch н”„лЎңм„ёмҠӨмһ…лӢҲлӢӨ.

кҙҖл Ё нҢҢлқјлҜён„°

# - Archiving -

#archive_mode = off             # enables archiving; off, on, or always
                                # (change requires restart)
#archive_command = ''           # command to use to archive a logfile segment
                                # placeholders: %p = path of file to archive
                                #               %f = file name only
                                # e.g. 'test ! -f /mnt/server/archivedir/%f && cp %p /mnt/server/archivedir/%f'
#archive_timeout = 0            # force a logfile segment switch after this
                                # number of seconds; 0 disables
  • archive_mode : WAL logfileмқ„ л°ұм—…н•ҳлҠ” archive_mode м„Өм • м—¬л¶Җ
  • archive_command : WAL лЎңк·ёлҘј л°ұм—…н•ҳкё° мң„н•ҙ мӢӨн–ү лҗ  лӘ…л №м–ҙ. %p : м•„м№ҙмқҙлёҢн•ҙм•јн•ҳлҠ” WAL logfileмқҳ кІҪлЎң, %f : нҢҢмқј мқҙлҰ„

AutoVacuum н”„лЎңм„ёмҠӨ

PostgreSQL мқҖ UPDATE, DELETE мһ‘м—…мқ„ мҲҳн–ү н•ң нӣ„ мқҙм „ лІ„м „мқҳ лҚ°мқҙн„°лҘј мҰүмӢң мӮӯм ңн•ҳм§Җ м•Ҡкі  л ҲмҪ”л“ңм—җ м ҖмһҘн•ҙ лҶ“мҠөлӢҲлӢӨ.
лҢҖмӢ , лҚ°мқҙн„°лҠ” PostgreSQLмқҳ MVCC л©”м»ӨлӢҲмҰҳм—җ мқҳн•ҙ мӮӯм ң лҗң кІғмңјлЎң н‘ңмӢңн•ҳкі  мқҙм „ лІ„м „мқҳ лҚ°мқҙн„°м—җ лӢӨлҘё нҠёлһңмһӯм…ҳмқҙ м•Ўм„ёмҠӨн•ҳлҠ” кІҪмҡ° мһ„мӢңлЎң мң м§Җн•©лӢҲлӢӨ.
нҠёлһңмһӯм…ҳмқҙ commit лҗң нӣ„м—җлҠ” мқҙм „ лІ„м „мқҳ лҚ°мқҙн„°к°Җ лҚ” мқҙмғҒ н•„мҡ”н•ҳм§Җ м•ҠмңјлҜҖлЎң (лҚ°л“ң нҠңн”Ң) PostgreSQLмқҖ кіөк°„ л°Ҹ xid нҷ•ліҙлҘј мң„н•ҙ лҚ°мқҙн„°лҘј м •лҰ¬н•ҙм•јн•©лӢҲлӢӨ.
мқҙ мһ‘м—…мқҖ AutoVacuum н”„лЎңм„ёмҠӨм—җм„ң мҲҳн–үлҗ©лӢҲлӢӨ.

кҙҖл Ё нҢҢлқјлҜён„°

#------------------------------------------------------------------------------
# AUTOVACUUM
#------------------------------------------------------------------------------

#autovacuum = on                        # Enable autovacuum subprocess?  'on'
                                        # requires track_counts to also be on.
#log_autovacuum_min_duration = -1       # -1 disables, 0 logs all actions and
                                        # their durations, > 0 logs only
                                        # actions running at least this number
                                        # of milliseconds.
#autovacuum_max_workers = 3             # max number of autovacuum subprocesses
                                        # (change requires restart)
#autovacuum_naptime = 1min              # time between autovacuum runs
#autovacuum_vacuum_threshold = 50       # min number of row updates before
                                        # vacuum
#autovacuum_analyze_threshold = 50      # min number of row updates before
                                        # analyze
#autovacuum_vacuum_scale_factor = 0.2   # fraction of table size before vacuum
#autovacuum_analyze_scale_factor = 0.1  # fraction of table size before analyze
#autovacuum_freeze_max_age = 200000000  # maximum XID age before forced vacuum
                                        # (change requires restart)
#autovacuum_multixact_freeze_max_age = 400000000        # maximum multixact age
                                        # before forced vacuum
                                        # (change requires restart)
#autovacuum_vacuum_cost_delay = 2ms     # default vacuum cost delay for
                                        # autovacuum, in milliseconds;
                                        # -1 means use vacuum_cost_delay
#autovacuum_vacuum_cost_limit = -1      # default vacuum cost limit for
                                        # autovacuum, -1 means use
                                        # vacuum_cost_limit
  • autovacuum : autovacuum н”„лЎңм„ёмҠӨлҘј мӮ¬мҡ©н• м§Җ м—¬л¶Җ
  • autovacuum_max_workers : autovacuum мқ„ лі‘л ¬лЎң мҲҳн–үн•  мөңлҢҖ м„ңлёҢ н”„лЎңм„ёмҠӨ мҲҳ м„Өм •
  • autovacuum_naptime : л‘җ autovacuum н”„лЎңм„ёмҠӨ мӮ¬мқҙмқҳ к°„кІ© мӢңк°„мқ„ м„Өм •
  • autovacuum_vacuum_threshold / autovacuum_analyze_threshold : н…Ңмқҙлё”м—җм„ң м—…лҚ°мқҙнҠё лҗң нҠңн”Ң мҲҳмқҳ мһ„кі„ к°’ м„Өм • , нҠңн”Ң м—…лҚ°мқҙнҠё мҲҳк°Җ мқҙ к°’мқ„ мҙҲкіјн•ҳл©ҙ vacuum / analyze мҲҳн–ү н•„мҡ”
  • autovacuum_vacuum_scale_factor / autovacuum_analyze_scale_factor : м„Өм •н•ң л°°мңЁл§ҢнҒј н…Ңмқҙлё” лӮҙ лҚ°мқҙн„° ліҖкІҪ мӢң, autovacuumмқҙ мҲҳн–үлҗ  н…Ңмқҙлё”лЎң м¶”к°ҖлҗЁ
  • autovacuum_freeze_max_age : н…Ңмқҙлё”мқҳ мөңлҢҖ мң нҡЁ xid, мқҙ к°’ мқҙмғҒмңјлЎң xidк°Җ м»Өм§Җм§Җ м•ҠлҸ„лЎқ vacuum мҲҳн–үн•Ё

нҶөкі„ мҲҳм§‘кё°

PostgreSQL optimizerмқҳ мҝјлҰ¬ мөңм Ғнҷ”лҘј мң„н•ҙ н…Ңмқҙлё” л©”нғҖм •ліҙ, мӮӯм ң / м—…лҚ°мқҙнҠё мҲҳ, лҚ°мқҙн„° лё”лЎқ мҲҳ, мқёлҚұмҠӨ ліҖкІҪ л“ұкіј к°ҷмқҖ лҚ°мқҙн„°лІ мқҙмҠӨ мһ‘м—… мӨ‘ нҶөкі„ м •ліҙлҘј мҲҳм§‘н•ҳлҠ” н”„лЎңм„ёмҠӨмһ…лӢҲлӢӨ.

кҙҖл Ё нҢҢлқјлҜён„°

#------------------------------------------------------------------------------
# STATISTICS
#------------------------------------------------------------------------------

# - Query and Index Statistics Collector -

#track_activities = on
#track_counts = on
#track_io_timing = off
#track_functions = none                 # none, pl, all
#track_activity_query_size = 1024       # (change requires restart)
#stats_temp_directory = 'pg_stat_tmp'
  • track_activities : м„ём…ҳм—җм„ң нҳ„мһ¬ мӢӨн–үмӨ‘мқё лӘ…л №м—җ лҢҖн•ң нҶөкі„ м •ліҙ мҲҳм§‘ кё°лҠҘмқ„ мӮ¬мҡ©н• м§Җ м—¬л¶Җ
  • track_counts : лҚ°мқҙн„°лІ мқҙмҠӨ лӮҙ лӘЁл“  мһ‘м—…м—җ лҢҖн•ң нҶөкі„ м •ліҙ мҲҳм§‘ кё°лҠҘмқ„ мӮ¬мҡ©н• м§Җ м—¬л¶Җ
  • track_io_timing : мҝјлҰ¬ мҲҳн–ү мӢң I/O мҶҢмҡ”мӢңк°„мқ„ мҲҳм§‘н•Ё
  • track_functions : н•ЁмҲҳ нҳём¶ң мҲҳ л°Ҹ мҲҳн–ү мӢңк°„ мҲҳм§‘
  • track_activity_query_size : к°Ғ м„ём…ҳм—җм„ң нҳ„мһ¬ мӢӨн–ү мӨ‘мқё м»Өл§Ёл“ңмқҳ н…ҚмҠӨнҠёлҘј м ҖмһҘн•ҳлҠ”лҚ° мӮ¬мҡ©н•  л©”лӘЁлҰ¬
  • stats_temp_directory : OS мғҒ нҶөкі„ м •ліҙмқҳ мһ„мӢң м ҖмһҘ кІҪлЎң

checkpointer

checkpointлҠ” DBMSк°Җ м„Өм •н•ҳлҠ” нҠёлһңмһӯм…ҳ нҸ¬мқёнҠёмһ…лӢҲлӢӨ. checkpointк°Җ мӢӨн–үлҗҳл©ҙ checkpoint мқҙм „мқҳ лӘЁл“  ліҖкІҪмӮ¬н•ӯмқҙ л””мҠӨнҒ¬лЎң flush лҗ©лӢҲлӢӨ.
DBMSк°Җ crash лҗң нӣ„ ліөкө¬н•  л•ҢлҸ„ checkpoint мқҙнӣ„ мӢңм җмқҳ WAL logлҘј л°ҳмҳҒн•ҳм—¬ ліөкө¬н•ҳкІҢлҗ©лӢҲлӢӨ.

кҙҖл Ё нҢҢлқјлҜён„°

# - Checkpoints -

#checkpoint_timeout = 5min              # range 30s-1d
max_wal_size = 1GB
min_wal_size = 80MB
#checkpoint_completion_target = 0.5     # checkpoint target duration, 0.0 - 1.0
#checkpoint_flush_after = 256kB         # measured in pages, 0 disables
