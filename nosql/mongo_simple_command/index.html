delete

  • 조건에 맞는 다큐먼트 1건 지우기
> db.TestCollection.remove({ "name": "super kimdubi"} )
WriteResult({ "nRemoved" : 10 })
  • 여러건 지우기
> db.TestCollection.remove ({ "name":"kimdubi"}, 2 )
WriteResult({ "nRemoved" : 0 })
  • 전부 지우기
> db.TestCollection.remove( {} )
WriteResult({ "nRemoved" : 4 })
