MySQL crash-safe replication

MySQL crash-safe replication кё°лҠҘмқҖ 5.6 лІ„м „л¶Җн„° мғқкёҙ мҳӨлһҳлҗң к°ңл…җмһ…лӢҲлӢӨ.
slave DBк°Җ crash лҗҳм–ҙ мһ¬кё°лҸҷ лҗҳлҠ” кІҪмҡ°м—җлҸ„ data мӨ‘ліөмқҙлӮҳ мң мӢӨ к°ҷмқҖ
replicationмқҙ к№Ём§ҖлҠ” кІҪмҡ°лҘј л§үкё° мң„н•ң кё°лҠҘмқёлҚ°мҡ”
мқҙлІҲ кёҖм—җм„ңлҠ” MySQL crash-safe replication м—җ лҢҖн•ҙм„ң мӮҙнҺҙліҙкІ мҠөлӢҲлӢӨ.

crash-safe replication к°ңл…җ

replication мҲҳн–ү мӨ‘ Slaveмқҳ statusлҘј fileм—җ update н•ҳкё° м „м—җ Slave DBк°Җ crash нӣ„ мһ¬кё°лҸҷ лҗҳлҠ” кІҪмҡ° л§Ҳм§Җл§ү нҠёлһңмһӯм…ҳмқ„ мһ¬мҲҳн–ү н•ҳкё° л•Ңл¬ём—җ мӨ‘ліө м—җлҹ¬л“ұмңјлЎң replicationмқҙ к№Ём§Ҳ мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙлҘј ліҙмҷ„н•ҳкё° мң„н•ҙ лӮҳмҳЁ м„Өм •мқҙ relay_log_info_repository=TABLE мқҙл©°
нҠёлһңмһӯм…ҳ м•Ҳм—җ replication м •ліҙлҘј нҸ¬н•ЁмӢңмјң data л°ҳмҳҒ SQLкіј replication status updateк°Җ н•ң нҠёлһңмһӯм…ҳмңјлЎң л¬¶м—¬
мқҙ л‘җк°Җм§Җк°Җ лӘЁл‘җ м„ұкіөн•ҙм•јл§Ң DBм—җ л°ҳмҳҒн•ҳлҸ„лЎқ н•ҳлҠ” м„Өм •мһ…лӢҲлӢӨ.

begin;
insert into tb_test values(1);
update mysql.slave_relay_log_info set Relay_log_name='./relay-bin.000006',replay_log_pos=1356;
commit;

м¶”к°ҖлЎң н•„мҡ”н•ң relay_log_recovery м„Өм •мқҖ Slave DB мһ¬кё°лҸҷ нӣ„ replication м„Өм •мқ„ л¶Ҳлҹ¬мҳ¬ л•Ң
master infoк°Җ м•„лӢҢ relay_log м •ліҙл§Ңмқ„ к°–кі  ліөкө¬лҘј мҲҳн–үн•ҳлҸ„лЎқ н•ҳлҠ” м„Өм • мһ…лӢҲлӢӨ.

м •лҰ¬н•ҳл©ҙ relay_log_info_repository=TABLE кіј relay_log_recovery м„Өм •мқ„ нҶөн•ҙ
Slaveк°Җ Crash лӮң нӣ„ мһ¬кё°лҸҷ мӢң л°ңмғқн•  мҲҳ мһҲлҠ” л¬ём ңм җл“Өмқ„ н•ҙкІ°н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

crash-safe replication process

мң„м—җм„ң мӮҙнҺҙліё crash-safe replicationмқҖ м•„лһҳмҷҖ к°ҷмқҖ нқҗлҰ„мңјлЎң лҸҷмһ‘н•©лӢҲлӢӨ.
л…№мғү н‘ңмӢңк°Җ crash-safe replicationм—җ м§Ғм ‘ кҙҖм—¬лҗҳлҠ” л¶Җл¶„мһ…лӢҲлӢӨ.

  • н•„мҲҳ м„Өм •
relay_log_info_repository = TABLE
relay_log_recovery = ON
  • replication process
1. I/O threadлҠ” Masterмқҳ binlog dataлҘј slaveмқҳ relay log лЎң м ҖмһҘ
2. I/O threadлҠ” к°Җм ёмҳЁ Masterмқҳ binlog м •ліҙлҘј mysql.slave_master_infoм—җ update
3. SQL threadлҠ” relay logлҘј мқҪм–ҙ transactionмқ„ slave DBм—җ л°ҳмҳҒн•Ё
4. SQL threadк°Җ л°ҳмҳҒн•ң relay log м •ліҙлҘј slave_relay_log_infoм—җ м—…лҚ°мқҙнҠён•Ё
