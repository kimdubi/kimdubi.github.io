PostgreSQL HOT update

PostgreSQLмқҳ MVCC лҠ” ORACLEмқҙлӮҳ MySQLмқҳ лҸҷмһ‘ л°©мӢқкіј лӢӨлҰ…лӢҲлӢӨ.

кіјкұ° лҚ°мқҙн„°лҘј лі„лҸ„мқҳ UNDO мҳҒм—ӯм—җ м ҖмһҘн•ҳлҠ” кІҢ м•„лӢҲлқј кіјкұ°мқҳ tupleмқ„ invalid мІҳлҰ¬ нӣ„ update нӣ„мқҳ мғҲлЎңмҡҙ tuple мқ„ м¶”к°Җн•ҳлҠ” л°©мӢқмңјлЎң лҸҷмһ‘н•ҳлҠ”лҚ°мҡ”

мһҘм җмңјлЎңлҠ”

  • UNDO мҳҒм—ӯмқ„ л”°лЎң кҙҖлҰ¬н•  н•„мҡ”к°Җ м—ҶлӢӨ
  • ROLLBACK мІҳлҰ¬кё° л§Өмҡ° л№ лҘҙлӢӨ

лӢЁм җмңјлЎңлҠ”

  • invalid лҗң мҳӨлһҳлҗң dead tupleл“Өмқ„ м •лҰ¬н•  vacuum мһ‘м—…мқҙ н•„мҡ”н•ҳлӢӨ
  • кіјлҸ„н•ң м—…лҚ°мқҙнҠёлЎң мқён•ң dead tuple лЎң н…Ңмқҙлё”мқҙ л№„лҢҖн•ҙ м§Ҳ мҲҳ мһҲлӢӨ
  • update лЎң мқён•ң мқёлҚұмҠӨ мҲҳм •мқҙ н•„мҡ”н•ҳлӢӨ

к°ҷмқҖ мһҘлӢЁм җмқҙ мһҲмҠөлӢҲлӢӨ.
к·ёлҰ¬кі  мқҙлҘј м–ҙлҠҗм •лҸ„ н•ҙмҶҢн•ҳкё° мң„н•ң л°©лІ•мқҙ HOT (Heap Only Tuple) м—…лҚ°мқҙнҠё л°©мӢқмқёлҚ°мҡ”

к·ёлҰјмІҳлҹј н…Ңмқҙлё” block мҷёл¶Җм—җм„ңлҠ” м–ҙл””м„ңлҸ„ м°ёмЎ°лҗҳм§Җ м•ҠлҠ” HOT tuple мқ„ мғқм„ұн•©лӢҲлӢӨ.
к·ёлҰ¬кі  мқҙ HOT tuple мқҳ мң„м№ҳлҠ” old versionмқҳ rowк°Җ к°–кі  мһҲлҠ”лҚ° л§Ҳм№ҳ pointer мІҳлҹј old versionмқҳ row, HOT chainмқ„ л”°лқј
мғҲлЎң м—…лҚ°мқҙнҠёлҗң HOT tuple м—җ лҸ„м°©н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мқҙ л°©лІ•м—җлҠ” лӢӨмқҢкіј к°ҷмқҖ мһҘм җмқҙ мһҲмҠөлӢҲлӢӨ.

  • мқёлҚұмҠӨлҘј мҲҳм •н•  н•„мҡ”к°Җ м—ҶлӢӨ => tuple мқҳ мҷёл¶Җ мЈјмҶҢлҠ” ліҖн•ң кІҢ м—Ҷкі  index лҠ” tupleмқҳ HOT chain мқ„ л”°лқјк°Җл©ҙлҗҳлӢҲк№Ң
  • dead tupleм—җ лҢҖн•ң vacuum л¶ҖлӢҙмқҙ м Ғм–ҙм§„лӢӨ.

мқҙ HOT tupleмқ„ мқҙмҡ©н•ң updateлҘј мқҙмҡ©н•ҳл Өл©ҙ л‘җк°Җм§Җ мЎ°кұҙмқ„ м¶©мЎұн•ҙм•јн•ҳлҠ”лҚ°

  • block м—җ кіөк°„мқҙ м¶©л¶„н•ҙм•јн•ңлӢӨ => old versionмқҳ tupleкіј HOT tuple мқҙ к°ҷмқҖ block м—җ мһҲм–ҙм•јн•Ё, м•„лһҳ к·ёлҰјмқҳ (a)
  • index key к°’м—җ ліҖкІҪмқҙ м—Ҷм–ҙм•ј н•ңлӢӨ => index key к°’мқҙ update лҗҳм–ҙ ліҖкІҪмқҙ мғқкё°л©ҙ мғҲлЎңмҡҙ мқёлҚұмҠӨ tupleмқҙ мғқм„ұлҗЁ,м•„лһҳ к·ёлҰјмқҳ (b)

н…ҢмҠӨнҠё

=> fillfactor лҠ” ORACLEмқҳ PCTFREEмқҳ к°ңл…җмңјлЎң blockмқ„ м–ҙлҠҗм •лҸ„ мӮ¬мҡ©н• м§Җ м •н•ҳлҠ” м„Өм •,
мң„м—җм„ңлҠ” н…Ңмқҙлё”мқҳ fillfactorлҘј 70%лЎң м„Өм •н•ҳм—¬ blockмқҳ 70%л§Ң мӮ¬мҡ©н•ҳкі  30%лҠ” updateлҘј мң„н•ҙ лӮЁкІЁл‘җкІҢ лҗңлӢӨ

old versionмқҳ tuple мқҙ update лҗң мғҲлЎңмҡҙ tupleмқҳ мң„м№ҳлҘј м•Ңл ӨмЈјлҠ” HOT chain л°©мӢқмқ„ мқҙмҡ©н•ҳкІҢлҗңлӢӨ

м •лҰ¬

postgresql мқҳ MVCC лҠ” UNDO м—Ҷмқҙ old version, new version лӘЁл‘җ к°–кі  мһҲлҠ” л°©мӢқмңјлЎң update н•  л•Ң л№„нҡЁмңЁмқҙ мһҲлҠ”лҚ°
мқҙлҘј fillfactor лҘј мқҙмҡ©н•ң HOT л°©мӢқмңјлЎң м–ҙлҠҗм •лҸ„ н•ҙмҶҢ н•  мҲҳ мһҲмҠөлӢҲлӢӨ. м°ёкі лЎң fillfactorлҠ” м „м—ӯліҖмҲҳк°Җ м•„лӢҲлқј н…Ңмқҙлё” к°ңлі„ м„Өм •мқҙкё° л•Ңл¬ём—җ
update к°Җ мһҰмқҖ н…Ңмқҙлё”мқҖ fillfactorлҘј мЎ°м Ҳн•ҳм—¬ HOT update мқҳ нҡЁкіјлҘј ліј мҲҳ мһҲмҠөлӢҲлӢӨ.
