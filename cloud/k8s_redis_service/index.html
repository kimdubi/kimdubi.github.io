Kubernetesм—җм„ң redis clusterлҘј лҸҢл Өліҙмһҗ - Service

Service?

Service к°ңл…җ

  • PodлҠ” deployment, replicaset, statefulset к°ҷмқҖ controllerм—җ мқҳн•ҙ кҙҖлҰ¬лҗҳкё° л•Ңл¬ём—җ мғҲлЎң мғқкІЁлӮҳкұ°лӮҳ м—Ҷм–ҙм§Җкё°лҸ„ н•ҳкі  н•ң Nodeм—җл§Ң л– мһҲлҠ” кІҢ м•„лӢҲлқј м—¬лҹ¬ NodeлҘј мҳ®кІЁлӢӨлӢҗ мҲҳ мһҲмқҢ
  • PodлҠ” мқҙл Үл“Ҝ лҸҷм ҒмңјлЎң ліҖн•ҳм—¬ кі м •лҗң endpointлЎңмқҳ нҳём¶ңмқҙ м–ҙл ӨмҡҙлҚ° (Podмқҳ IP,hostnameмқҙ ліҖн•ҳлӢҲк№Ң ) мқҙлҹ¬н•ң мқҙмҠҲлҘј н•ҙкІ°н•ҳкё° мң„н•ң мҳӨлёҢм қнҠёк°Җ Serviceмһ„
  • ClientлҠ” нҒҙлҹ¬мҠӨн„° м•Ҳм—җм„ң Pod IPк°Җ л°”лҖҢлҚҳ м—Ҷм–ҙм§Җкұҙ мғҲлЎң мғқкё°кұҙ мӢ кІҪм“ё н•„мҡ” м—Ҷмқҙ Serviceл§Ң л°”лқјліҙл©ҙ кі м •лҗң endpointлЎң м ‘к·јмқҙ к°ҖлҠҘн•Ё
  • ServiceлҠ” нҒ¬кІҢ м•„лһҳмҷҖ к°ҷмқҙ л„Өк°Җм§Җ TypeмңјлЎң лӮҳлүЁ
    • ClusterIp : kubernets cluster лӮҙл¶Җм—җм„ң мӮ¬мҡ©н•ҳлҠ” service. мҷёл¶Җм—җм„  м ‘к·јн•  мҲҳ м—ҶмқҢ
    • LoadBalancer : нҶ нҒҙ,aws,gcp л“ұ нҚјлё”лҰӯ нҒҙлқјмҡ°л“ң м„ңл№„мҠӨлҘј мқҙмҡ©н•  л•Ң мӮ¬мҡ©к°ҖлҠҘн•ң typeмңјлЎң loadbalancerмқҳ IPлҘј мқҙмҡ©н•ҳм—¬ мҷёл¶Җм—җм„ң м ‘к·јк°ҖлҠҘн•Ё
    • NodePort : kubernetes clusterмқҳ к°Ғ л…ёл“ңмқҳ м§Җм •лҗң portлҘј serviceм—җ н• лӢ№н•Ё. nodeмқҳ portлҘј мқҙмҡ©н•ҳкё° л•Ңл¬ём—җ мҷёл¶Җм—җм„ңлҸ„ м ‘к·јк°ҖлҠҘн•Ё
    • ExternalName : kubernetes cluster лӮҙл¶Җм—җм„ң мҷёл¶ҖлЎң м ‘к·јн•ҳлҠ” кІҪмҡ° мӮ¬мҡ©, ServiceлҘј ExternalName мңјлЎң м„Өм •н•ҳм—¬ CNAMEмқ„ нғҖкі  мҷёл¶ҖлЎң м ‘к·јн•Ё
  • мқҙ кёҖм—җм„ң лӢӨлЈЁлҠ” redis-cluster м—җм„ңлҠ” ClusterIP typeмқҳ serviceлҘј мӮ¬мҡ©н•ҳкі  мһҲмқҢ ( DBлҠ” лӮҙл¶ҖнҶөмӢ л§Ң н•ҳл©ҙ лҗҳлӢҲк№Ң)

headless serviceлһҖ?

  • kimdubi-test-redis-cluster к°ҷмқҖ ліҙнҶөмқҳ serviceлҠ” л°ӣмқҖ мҡ”мІӯмқ„ л§Өн•‘лҗң Podл“Өм—җкІҢ loadbalancing нҳ•нғңлЎң лҝҢл ӨмЈјлҠ”лҚ° мқҙлҠ” stateless н•ң web server мІҳлҹј, м–ҙл–Ө web serverлЎң мҡ”мІӯмқҙ к°ҖлҸ„ лҸҷмқјн•ң кІ°кіјлҘј лӮҙлҠ” кө¬мЎ°м—җм„ң мқҳлҜёк°Җ мһҲмқҢВ 
  • DBмІҳлҹј Master / Slave к°ҷмқҖ stateful н•ң кө¬м„ұм—җм„ңлҠ” loadbalancing нҳ•нғңлЎң лҝҢл ӨмЈјлҠ” кІҢ м•„лӢҲлқј Master мқём§Җ Slaveмқём§Җ нҢҢм•…н•ҳм—¬ нҠ№м •н•ң Podм—җ мҡ”мІӯмқ„ лҝҢл ӨмӨҳм•јн•Ё
  • мқҙмІҳлҹј loadbalancingмқҙ н•„мҡ”м—Ҷкі  client м—җм„ң лӘЁл“  Podм—җ м ‘к·јн•ҙм•јн•ҳлҠ” кө¬м„ұм—җм„ңлҠ” headless serviceлҘј мӮ¬мҡ©н•ҳл©° м•„лһҳмҷҖ к°ҷмқҖ нҠ№м§•мқҙ мһҲмқҢ
    • headless serviceлҠ” IP лҢҖмӢ  dns лҘј к°Җм§җ
    • headless serviceмқҳ dnsм—җ лҢҖн•ҙ nslookupмқ„ мҲҳн–үн•ҳл©ҙ binding лҗң лӘЁл“  Podл“Өмқҳ IPлҘј return н•Ё
    • ClientлҠ” return л°ӣмқҖ лӘЁл“  Podмқҳ IPлҘј нҶөн•ҙ мӣҗн•ҳлҠ” Pod м—җ м§Ғм ‘ м ‘к·јн•  мҲҳ мһҲмқҢ
